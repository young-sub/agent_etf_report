# Triage Labels

Labels are not yet verified in GitHub.

Recommended vocabulary until labels are configured:

- `type:plan`
- `type:docs`
- `type:migration`
- `type:bug`
- `type:feature`
- `area:packaging`
- `area:domain`
- `area:document-evidence`
- `area:compatibility`
- `area:agent-pack-cleanup`
- `status:blocked`
- `status:ready`

Do not create, rename, or delete remote labels without explicit approval.
